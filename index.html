The Logic and Learning (LOL) lab works on logic and machine learning, i.e. Inductive Logic Programming (ILP) at the University of Oxford. We developed the ILP system Popper

We will recruit multiple postdocs in 2021, 2022, and 2023 to work on the EPSRC-funded project The Automatic Computer Scientist. If interested, please see the join page.
