Logic and Learning lab

About People Join

The Logic and Learning (LOL) lab is in the computer science department at the University of Oxford. We work on program synthesis: using machine learning to write computer programs. We focus on inductive logic programming (ILP), which combines logic and machine learning. We maintain the ILP system Popper.

We will recruit multiple postdocs in 2022, 2023, and 2024 to work on the EPSRC-funded project The Automatic Computer Scientist. If interested, please see the join page.
